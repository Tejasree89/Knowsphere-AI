# KnowSphere AI

An Enterprise Knowledge Platform powered by AI and Retrieval-Augmented Generation (RAG).

## Features

- Multi-tenant architecture
- Company-based authentication
- Document upload (PDF, DOCX, TXT)
- AI-powered question answering
- Secure chat history
- Admin dashboard
- Role-based access control

## Tech Stack

### Backend
- FastAPI
- PostgreSQL
- SQLAlchemy
- LangChain
- ChromaDB

### Frontend
- React
- Tailwind CSS

### AI
- OpenAI API
- Sentence Transformers
